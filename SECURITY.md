# Security Policy

## Supported Versions

Security fixes are applied to the latest released version.

| Version | Supported |
| --- | --- |
| Latest | Yes |
| Older releases | No |

## Reporting a Vulnerability

Please do not report security vulnerabilities in public GitHub issues.

Send details to: **abubakar1808031@gmail.com**

Include:

- Description of the vulnerability
- Affected version(s)
- Reproduction steps or proof of concept
- Potential impact

## Response Targets

- Initial acknowledgment: within 72 hours
- Triage decision: within 7 days
- Status updates: at least weekly until resolution

## Disclosure

After a fix is available, we will coordinate responsible disclosure and publish release notes with mitigation details.
