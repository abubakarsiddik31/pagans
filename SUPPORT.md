# Support

## Getting Help

For questions about usage, setup, or integration:

- Open a GitHub Discussion (preferred for Q&A)
- Open a GitHub Issue for confirmed bugs or actionable enhancements

## What to Include

- PAGANS version
- Python version
- Operating system
- Minimal reproducible snippet
- Relevant logs and traceback

## Response Expectations

This is an open-source project maintained on a best-effort basis. Response times may vary.

## Security

For security issues, do not open a public issue. See [SECURITY.md](SECURITY.md).
